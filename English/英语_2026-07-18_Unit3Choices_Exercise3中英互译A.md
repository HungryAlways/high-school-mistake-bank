## 📝 错题 #1.1（Q1）

### 【题目原文】
衣服标签上有品牌名、尺寸和售价。（there be）

### 【你的答案】
**There are the name of, size and sellings on the clothes' label.**

### 【正确答案】
**On the clothes' label, there are the brand name, size and price.**

### 【错误原因分析】
1. **词汇错误**：
   - "品牌名" ≠ "name of" → 应为 **brand name**（复合名词）
   - "售价" ≠ "sellings"（无此词）→ 应为 **price**
2. there be 句型正确使用，但顺序上"On the label, there are..."更地道

### 【涉及知识点】
- **brand name** 品牌名（固定搭配）
- **price / sale price** 售价
- **label** 标签（clothes' label 衣服标签）

## 📝 错题 #1.2（Q2）

### 【题目原文】
光的传播速度很快，所以古人无法测量它的速度。（travel）

### 【你的答案】
**The spread of traveling of the light is so quick, so ancient people couldn't measure its speed.**

### 【正确答案】
**Light travels so quickly that ancient people couldn't measure its speed.**

### 【错误原因分析】
1. **中式思维直译**："光的传播速度" 直接翻译成 "the spread of traveling of the light"，冗余且错误
2. **travel 用作动词**：题目提示词 travel 本身就是"传播、行进"的意思，直接作谓语用
3. **so...that... 句型**：表因果，"太快以至于..."
4. **副词 quickly**：修饰动词 travel，不用 quick

### 【涉及知识点】
- **travel (v.)** 传播、行进（Light travels at 300,000 km/s.）
- **so + adv/adj + that...** 句型
- 中式思维要转成英语思维：一个句子一个主谓

## 📝 错题 #1.3（Q3）

### 【题目原文】
这个雕像很值钱，不是因为它尺寸巨大，而是因为它的制造可以追溯到唐朝。（not because...but because...）

### 【你的答案】
**This statue is valuable, not because it's huge in size, but because its measure can date back to Tang Dynasty.**

### 【正确答案】
**This statue is valuable, not because it's huge in size, but because its production can date back to / dates back to the Tang Dynasty.**

### 【错误原因分析】
1. **词汇错误**："制造" 不是 measure（测量）→ 应为 **production / making**
2. **date back to 主动语态**（左边旁批已写！）：
   - **date back to / date from** = 主动，主语是"东西/事物"
   - **be traced back to** = 被动，主语可以是"被追溯的对象"
3. **定冠词**：the Tang Dynasty（朝代名前加 the）

### 【涉及知识点】
- **date back to / date from** = 追溯到（主动，无被动形式！）
- **trace...back to / be traced back to** = 追溯（可被动）
- 朝代前加 the：the Tang Dynasty, the Ming Dynasty

## 📝 错题 #1.4（Q4）

### 【题目原文】
有朋自远方来，不亦乐乎！（all the way）

### 【你的答案】
**Friends come all the way is happy.**

### 【正确答案】
**How happy it is to have friends come all the way from afar!**

### 【错误原因分析】
1. **感叹句句式**：中文"不亦乐乎"是感叹，英语要用 **How + adj + it is...** 感叹句
2. **from afar** = 从远方（副词，afar 是"远方"的意思）
3. **have sb. do sth.** = 让某人做某事
4. 中式直译错误：把"来是快乐的"直接翻译成 "come is happy"，句法崩坏

### 【涉及知识点】
- **How + 形容词 + it is + to do sth.**（感叹句型）
- **from afar** = from far away 从远方
- **all the way** = 一路上、大老远
- **have sb. do sth.** 让某人做某事

## 📝 错题 #1.5（Q5）⚠️ 重灾区

### 【题目原文】
一些建筑材料已被运输到那个已经关闭的景点，那里有许多历史遗迹需要保护。（need）

### 【你的答案】
**Some materials of architecture are transported to the scenespot that had shut, there are many historic sites needing to protected.**

### 【正确答案】
**Some building materials have been transported to the tourist attraction that has been closed, where there are many historic sites needing to be protected.**

### 【错误原因分析】
1. **词汇错误**：
   - "建筑材料" ≠ materials of architecture → 应为 **building materials**
   - "景点" ≠ scenespot（无此词）→ 应为 **tourist attraction / scenic spot**
2. **时态错误**："已被运输" = 现在完成时被动语态 → **have been transported**
3. **被动语态错误**："已经关闭" → **has been closed**（被关闭）
4. **need doing/to be done**："需要被保护" 两种表达：
   - needing to **be protected**（被动不定式）
   - needing **protecting**（动名词主动表被动）
   - 你写的 "needing to protected" 少了 be，语法崩坏
5. **定语从句**：用 where 引导"那里有..."，不能用 there are 逗号连接（run-on sentence）

### 【涉及知识点】
- **building materials** 建筑材料
- **tourist attraction / scenic spot** 景点
- **现在完成时被动语态**：have/has been + 过去分词
- **need to be done / need doing**：需要被做（重点句型！）
- **where 引导的定语从句**：先行词是地点

## 📝 错题 #1.6（Q6）

### 【题目原文】
上海的垃圾分类规则很严格，驱动一些公司创出新的业务，例如：帮居民倒垃圾。（dump）

### 【你的答案】
**The rules of sorting rubbish in Shanghai is very strict, driving some businesses create new businesses such as helping residents dump rubbish.**

### 【正确答案】
**The rules of classifying / sorting rubbish in Shanghai are very strict, driving some companies to create new business, such as helping residents dump rubbish.**

### 【错误原因分析】
1. **主谓一致**：主语是 The rules（复数），谓语用 **are**（不是 is）
2. **词汇混淆**：
   - "公司" = **companies**（不是 businesses）
   - "业务" = **business**（不可数）
3. **drive sb. to do sth.**：驱动某人做某事，需加 **to**
   - 你漏掉了 to → driving some companies **to** create...

### 【涉及知识点】
- **主谓一致**：The rules ARE strict（rules 复数）
- **drive sb. to do sth.** 驱使某人做某事
- **company（公司） vs business（生意/业务）**：
  - company 可数，指公司实体
  - business 抽象概念时不可数（do business, create new business）
- **classify / sort rubbish** 垃圾分类
- **dump rubbish** 倒垃圾

## 📝 错题 #1.7（Q7）⚠️ 重灾区

### 【题目原文】
很多东西通常是热胀冷缩，但水却是反过来的。（heat）

### 【你的答案】
**Many things always be heat when they expend big and freeze to small, but water is opposite to.**

### 【正确答案】
**Many things usually expand when heated and contract when cooled, but water is quite the opposite.**

### 【错误原因分析】
1. **核心词汇错误**：
   - "膨胀" = **expand**（不是 expend "花费"！）⚠️ 拼写混淆
   - "收缩" = **contract**（不是 freeze "冻结" 或 shrink to small）
2. **被动分词**：when (they are) **heated** / when (they are) **cooled**
   - 时间状语从句省略主谓
3. **quite the opposite** = 恰恰相反（固定表达）
   - 你写的 "is opposite to" 不完整，缺宾语

### 【涉及知识点】
- **expand 膨胀 ↔ contract 收缩**（物理学核心词，必背！）
- **expend vs expand**：expend = 花费，expand = 膨胀（拼写易混！）
- **when + 过去分词**（省略"they are"）：when heated, when cooled
- **quite the opposite** = 恰好相反（=the other way around）
- **热胀冷缩**：expand when heated and contract when cooled

## 📝 错题 #1.8（Q8）

### 【题目原文】
如果要建设一座生态友好型城市，我们必须减少塑料容器和包装纸的使用，并且同时建立充足的回收设施。（if）

### 【你的答案】
**If wants to build one eco-friendly city, we must reduce, the use of plastic containers and package paper, and build site up recycle facilities.**

### 【正确答案】
**If we want to build an eco-friendly city, we must reduce the use of plastic containers and packaging paper, and at the same time build enough recycling facilities.**

### 【错误原因分析】
1. **主语缺失**：If **we** want（漏了主语 we）
2. **冠词**：an eco-friendly city（eco 元音开头，用 an，不用 one）
3. **动名词作形容词**：
   - **packaging paper** 包装纸（package → packaging）
   - **recycling facilities** 回收设施（recycle → recycling）
4. **漏译**："并且同时" = **and at the same time**（漏了）
5. **enough** 位置：enough recycling facilities（enough 修饰名词放前面）

### 【涉及知识点】
- **动名词作形容词（超高频）**：
  - packaging paper 包装纸
  - recycling facilities 回收设施
  - swimming pool 游泳池
  - shopping mall 购物中心
- **an eco-friendly city**（eco 元音开头）
- **at the same time** 同时

## 📝 错题 #1.9（Q9 英译中）

### 【题目原文】
Compared with the fruits and vegetables that grow in greenhouses, those growing in natural sunlight are more nutritious and their production process is more energy-efficient.

### 【你的答案】
**未作答**

### 【正确答案】
**与生长在温室里的水果和蔬菜相比，那些在自然阳光下生长的（水果和蔬菜）更有营养，而且它们的生产过程更加节能。**

### 【重点词汇 & 短语】

- **greenhouse** 温室
- **nutritious** (adj.) 有营养的
- **energy-efficient** (adj.) 节能的
- **those growing in...** = those that grow in...（现在分词作后置定语）

### 【翻译技巧】
1. **过去分词开头**：Compared with... 翻译时提到句首"与...相比"
2. **定语从句/分词**：翻译成"...的..."前置
3. **energy-efficient** 是复合形容词，译为"节能的"

## 🎯 本大题错点归纳

### 一、必背固定搭配 & 词汇

| 类别 | 关键短语 |
|---|---|
| **词汇** | brand name(品牌名), building materials(建筑材料), tourist attraction(景点), classify rubbish(垃圾分类), carbon neutrality(碳中和) |
| **动词** | travel(传播), expand ↔ contract(膨胀↔收缩), date back to(追溯到) |
| **句型** | so...that..., How + adj + it is to do..., need to be done, drive sb. to do sth. |
| **动名词形容词** | packaging paper, recycling facilities, building materials |

### 二、易错点 TOP 5

1. **主谓一致**：The rules **are** strict.（rules 复数）
2. **被动语态**：have been transported / has been closed / need to be protected
3. **动名词作形容词**：packaging(不是 package), recycling(不是 recycle)
4. **英语单动词表达**：Light **travels**（不要写 the traveling of light）
5. **中式思维**：避免逐字翻译，先想句子主干（主谓宾）

### 三、易混拼写

- **expand**（膨胀） vs **expend**（花费）
- **contract**（收缩） vs **contact**（联系）
- **classify**（分类） vs **classical**（古典的）
