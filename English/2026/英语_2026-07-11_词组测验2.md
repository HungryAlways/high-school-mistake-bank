## 错题 #1

**学科**: 英语 | **章节**: 词组搭配（第3题）

### 【题目原文】
3. 赞成 / 反对

### 【你的答案】
`agree for / against`

### 【正确答案】
`agree with / disagree with（反对）；be for / against（赞成/反对）`

### 【错误原因分析】
"赞成"不用 agree for。表态度：agree with sb.（同意某人）；表"赞成/反对"最地道是 **be for / be against**。agree 后不接 for。

### 【涉及知识点】
- agree with / agree to / agree on
- be for（赞成）vs be against（反对）

### 【题目原文】
8. 申请担任秘书

### 【你的答案】
`apply to become secertary`

### 【正确答案】
`apply for a position as a secretary`

### 【错误原因分析】

### 【涉及知识点】
- apply for a position 申请职位
- apply to sb / apply to do 辨析

## 错题 #3

**学科**: 英语 | **章节**: 句型（第10题）

### 【题目原文】
10. （在某人看来）似乎

### 【你的答案】
`seem to do`

### 【正确答案】
`It appears to sb. that... / It seems to sb. that...`

### 【错误原因分析】
题目要"（在某人看来）似乎"这个句型，应用 **It appears/seems to sb. that...**。appear 是实义动词，前面不加 is。

### 【涉及知识点】
- It appears/seems (to sb.) that...
- appear 作系动词/实义动词

## 📌 高频陷阱小结

**① 介词搭配（本卷最大失分点）**
| 短语 | 正确介词 |
|---|---|
| ask sb. **about** sth. | 询问 |
| ask sb. **for** sth. | 要求得到 |
| apply **for** a position | 申请 |
| apply sth. **to** sth. | 应用于 |
| be angry **with** sb. / **at** sth. | 生气 |
| have an appetite **for** | 渴望 |
| arrange **for** sb. to do | 安排 |
| arise **from** | 由…产生 |

**② 固定句型**
- It appears/seems to sb. that...
- I would appreciate it if you would...
- on one's arrival = as soon as one arrives

**③ 短语词义（英译汉易错）**
- allow for = 考虑到（≠允许）
- appeal to = 呼吁/吸引
- above all = 最重要的是（≠之前）

**④ 拼写**
- secretary / appetite / appliance
