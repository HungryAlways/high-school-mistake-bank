# High School Mistake Bank

高中错题库，按学科、年份与间隔复习节点整理。

## 使用方法

1. 从 [`_templates/mistake-template.md`](_templates/mistake-template.md) 复制错题模板。
2. 将错题放入对应学科和年份目录。
3. 在 [`INDEX.md`](INDEX.md) 登记错题。
4. 按 [`REVIEW.md`](REVIEW.md) 的 Day 1、Day 3、Day 7、Day 14、Day 30 节奏复习。
5. 在 [`STATISTICS.md`](STATISTICS.md) 更新统计。

每个错题文件都必须使用 YAML Front Matter。
